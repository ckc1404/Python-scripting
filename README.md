When path gets distorted :  export PATH=/bin:/usr/bin:/usr/local/bin:/sbin:/usr/sbin

```py
sudo nano ~/.bashrc
PASTE at the end of file:

export PATH="/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:bin:/usr/games:/usr/local/games"
ctrl+X -> Y -> Enter
source ~/.bashrc
```
